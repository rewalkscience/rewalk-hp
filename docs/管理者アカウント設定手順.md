# 管理者アカウント設定手順（ゆうたろう向け）

## 1. 会員登録
サイトの「新規登録」からアカウントを作成する（rewalk.science@gmail.com 推奨）。

## 2. 管理者に昇格（初回のみ・田村が実行）
```
cd api
npx wrangler d1 execute rewalk-db --remote --command "UPDATE users SET role='admin' WHERE email='rewalk.science@gmail.com'"
```

## 3. 再ログイン
一度ログアウトして再ログインすると、管理ダッシュボード（admin-dashboard.html）に自動遷移する。

## 4. 2人目以降の管理者
管理画面の「会員一覧」から「管理者にする」ボタンで昇格できる（SQLは不要）。

## Stripe設定（Cloudflareダッシュボードから）
Workers & Pages → rewalk-api → Settings → Variables and Secrets で以下を設定:
- `STRIPE_SECRET_KEY` : Stripeダッシュボードの「開発者 > APIキー > シークレットキー」
- `STRIPE_WEBHOOK_SECRET` : Webhook署名シークレット

### Stripe Webhook URL（Stripeダッシュボードに登録）
```
https://rewalk-api.rewalk-science.workers.dev/api/webhook/stripe
```
イベント: `checkout.session.completed` のみでOK。

※ 旧手順書（stripe-key-setup.docx）に記載のURL（rewalk-api.tamu-ne-co-jp.workers.dev）は無効。上記URLを使うこと。
